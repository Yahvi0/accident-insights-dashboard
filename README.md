# accident-insights-dashboard
An interactive Power BI dashboard for analyzing road accident data, identifying trends, and generating actionable safety insights.

## Overview
This dashboard analyzes a dataset containing details of reported road accidents, including accident severity, number of vehicles involved, casualties, geographic location, road conditions, weather conditions, and police response data. The objective is to create an interactive dashboard that visualizes accident trends, identifies high-risk areas, and helps in data-driven decision-making for traffic management and safety improvements.

## Tools Used
- Power BI
- Excel
- DAX

##  Objectives
- Analyze overall road accident trends.
- Monitor total accidents and casualties.
- Compare accident severity levels.
- Identify high-risk road and weather conditions.
- Analyze accidents by vehicle type.
- Provide interactive filtering for detailed analysis

## Features
- KPI tracking
- Region-wise analysis
- Interactive Slicers and Filters
